# Public Health Dashboard Prototype

This project is a demo of a unified public health portal concept for Maharashtra.

The idea behind the prototype is simple: public health officials currently depend on many different applications and portals across Government of India and Government of Maharashtra systems. This creates fragmentation, repeated navigation, and difficulty in getting a consolidated view.

This prototype explores how a single access layer with dashboard based navigation could make the system easier to use for senior officials and program teams.

## Live Demo

[View the live dashboard prototype](https://phddashboard.netlify.app/)

## Problem

In Maharashtra public health, more than 40 digital applications and portals are used across different programs and administrative levels. These systems serve important functions, but they are often accessed separately.

This creates challenges such as:

- Fragmented access across multiple portals
- Difficulty in viewing program progress in one place
- Repeated login and navigation burden
- Limited senior leadership visibility across systems

## Proposed Concept

The prototype presents a unified public health portal where different health applications, dashboards, and program systems can be accessed from one place.

The larger idea is to enable:

- Single point access to multiple systems
- Dashboard based monitoring
- Faster navigation for officials
- Better visibility for senior decision makers
- Foundation for single sign on in the future
